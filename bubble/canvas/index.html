BubbleCanvas

@nivo/circle-packinghierarchycanvas
svghtmlcanvashttp api

A variation around the Bubble component. Well suited for large data sets as it does not impact DOM tree depth, however you'll lose the isomorphic ability and transitions (for now).

The responsive alternative of this component is ResponsiveBubbleCanvas. It also offers various implementations, see Bubble and BubbleHtml.
