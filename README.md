**nivo** provides supercharged React components to easily build dataviz apps,
it's built on top of d3.

Several libraries already exist for React d3 integration,
but just a few provide server side rendering ability and fully declarative charts.

## Installation

In order to use nivo, you just have to pick the scoped `@nivo` packages according to the charts you wish to use.

```
yarn add @nivo/bar @nivo/sankey ...
```

## Features

-   Highly customizable
-   Motion/transitions, powered by [react-motion](https://github.com/chenglou/react-motion)
-   [Component playground](https://nivo.rocks)
-   [Exhaustive documentation](https://nivo.rocks)
-   Isomorphic rendering
-   [SVG charts](https://nivo.rocks/components?filter=svg)
-   [HTML charts](https://nivo.rocks/components?filter=html)
-   [Canvas charts](https://nivo.rocks/components?filter=canvas)
-   [server side rendering API](https://github.com/plouc/nivo-api)
-   [SVG patterns](https://nivo.rocks/guides/patterns)
-   [Gradients](https://nivo.rocks/guides/gradients)
-   [responsive charts](https://nivo.rocks/components?q=responsive)
